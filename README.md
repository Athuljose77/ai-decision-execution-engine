# AI-Powered Group Decision & Execution Engine

## Overview

This project is an AI-powered collaboration system that monitors group discussions, detects ideas, identifies consensus, and automatically generates structured execution plans.

It helps student teams, developers, and hackathon participants move efficiently from brainstorming to execution.

## Problem

Teams often struggle to identify the best idea, reach consensus, and convert discussions into actionable plans.

## Solution

This system introduces an AI engine that:

- Detects ideas automatically
- Clusters similar ideas
- Identifies consensus
- Generates execution roadmap
- Suggests tasks, timeline, and tech stack
